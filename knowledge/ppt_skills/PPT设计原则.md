# PPT 设计原则

## 核心理念

> 一页PPT只传达一个核心观点，让观众在30秒内理解内容。

---

## 五大设计原则

### 1️⃣ 简洁原则 (Simplicity)

**核心要义：** 删繁就简，突出重点

**实践方法：**
- 每页不超过 3-5 个要点
- 每个要点不超过一行
- 删除装饰性元素
- 留白也是一种设计

```
❌ 错误示范：
• 这是一个很长的要点，包含了太多细节
• 另一个也很长的要点，观众根本记不住
• 还有第三个超长的要点，信息过载了

✅ 正确示范：
• 核心观点一
• 核心观点二  
• 核心观点三
```

---

### 2️⃣ 聚焦原则 (Focus)

**核心要义：** 引导观众视线，突出核心信息

**视觉层次：**
```
┌─────────────────────────────────────┐
│                                     │
│         【标题】- 最醒目             │
│                                     │
│    【核心观点】- 次醒目              │
│                                     │
│  • 支撑论点 1                       │
│  • 支撑论点 2                       │
│  • 支撑论点 3                       │
│                                     │
│    【辅助信息】- 最不醒目            │
│                                     │
└─────────────────────────────────────┘
```

**实现方法：**
- 标题字号最大 (≥32pt)
- 核心内容次之 (≥24pt)
- 辅助信息最小 (≥18pt)
- 用颜色、粗细区分层次

---

### 3️⃣ 一致性原则 (Consistency)

**核心要义：** 统一风格，专业美观

**需要统一的元素：**

| 元素 | 统一要点 |
|------|---------|
| 字体 | 全文使用同一字体族 |
| 颜色 | 使用统一的配色方案 |
| 图标 | 风格一致，大小统一 |
| 排版 | 相同内容用相同布局 |
| 图片 | 统一滤镜或风格 |

**推荐字体：**
- 标题：微软雅黑 Bold / 思源黑体 Bold
- 正文：微软雅黑 / 思源黑体 Regular
- 英文：Arial / Helvetica / Open Sans

---

### 4️⃣ 可视化原则 (Visualization)

**核心要义：** 用图表代替文字，一目了然

**数据可视化选择：**

| 数据类型 | 推荐图表 |
|---------|---------|
| 占比关系 | 饼图、环形图 |
| 对比关系 | 柱状图、条形图 |
| 趋势变化 | 折线图、面积图 |
| 相关关系 | 散点图、气泡图 |
| 流程步骤 | 流程图、时间轴 |
| 层级结构 | 树状图、组织架构图 |

**可视化原则：**
- 图表要能独立表达含义
- 标题、图例、数据标签齐全
- 颜色数量 ≤ 5 种
- 突出重点数据

---

### 5️⃣ 对齐原则 (Alignment)

**核心要义：** 规范对齐，整洁有序

**对齐规则：**
```
┌─────────────────────────────────────┐
│                                     │
│  ←─────── 左对齐 ───────→           │
│                                     │
│  标题                               │
│  ├─ 要点一                          │
│  ├─ 要点二                          │
│  └─ 要点三                          │
│                                     │
│  图表元素也遵循对齐规则              │
│  ┌─────┐ ┌─────┐ ┌─────┐           │
│  │  A  │ │  B  │ │  C  │           │
│  └─────┘ └─────┘ └─────┘           │
│                                     │
└─────────────────────────────────────┘
```

**网格系统：**
- 使用 12 列网格
- 元素对齐网格线
- 保持边距一致

## 排版技巧

### 黄金比例
- 标题区域：内容区域 = 1 : 1.618
- 留白区域占页面 30-40%

### 字号建议
- 标题：36-48pt
- 正文：24-28pt
- 注释：18-20pt
- 最小不低于 16pt

### 行间距
- 标题行间距：1.2
- 正文行间距：1.5
- 列表行间距：1.8

## 常见错误

| 错误 | 问题 | 改进 |
|------|------|------|
| 文字堆砌 | 信息过载 | 精简要点，分页展示 |
| 字体过多 | 风格混乱 | 最多使用 2 种字体 |
| 颜色杂乱 | 视觉干扰 | 限制在 3-5 种颜色 |
| 图片模糊 | 不专业 | 使用高清图片 |
| 动画过多 | 分散注意力 | 简洁过渡即可 |
