## 常用图表类型

### 1. 柱状图 (Bar Chart)

**适用场景：**
- 比较不同类别的数值
- 展示排名
- 显示变化幅度

**设计要点：**
```
✅ 正确做法：
┌─────────────────────────┐
│                         │
│   ▓▓▓▓▓                 │
│   ▓▓▓▓▓  ▓▓▓▓           │
│   ▓▓▓▓▓  ▓▓▓▓  ▓▓▓      │
│   ▓▓▓▓▓  ▓▓▓▓  ▓▓▓  ▓   │
│   ─────────────────────  │
│    A     B     C    D   │
└─────────────────────────┘

- 基线从零开始
- 柱子间距一致
- 颜色有区分度

❌ 常见错误：
- Y轴不从零开始（夸大差异）
- 柱子过多（超过8个难对比）
- 3D效果（视觉误导）
```

---

### 2. 折线图 (Line Chart)

**适用场景：**
- 展示趋势变化
- 时间序列数据
- 连续数据变化

**设计要点：**
```
✅ 正确做法：
┌─────────────────────────┐
│     •───•───•           │
│    /         \          │
│   •           •───•     │
│  /                 \    │
│ •                   •   │
│ ─────────────────────── │
│  Q1  Q2  Q3  Q4  Q5    │
└─────────────────────────┘

- 线条数量 ≤ 4 条
- 标注关键数据点
- 清晰的图例

❌ 常见错误：
- 线条过多，难以区分
- 时间轴方向错误
- 省略坐标轴标签
```

---

### 3. 饼图 (Pie Chart)

**适用场景：**
- 展示占比关系
- 少量分类（≤5个）
- 强调某一类别

**设计要点：**
```
✅ 正确做法：
┌─────────────────────────┐
│           ┌───┐         │
│       ┌───│ A │───┐     │
│       │   └───┘   │     │
│       │    35%    │     │
│   ┌───┴───────────┴───┐ │
│   │        B          │ │
│   │       45%         │ │
│   └───────────────────┘ │
│         C (20%)         │
│                         │
│  A: 产品线A 35%         │
│  B: 产品线B 45%         │
│  C: 产品线C 20%         │
└─────────────────────────┘

- 分类数量 ≤ 5
- 按大小排序
- 标注具体数值/百分比

❌ 常见错误：
- 分类过多（用条形图代替）
- 使用3D饼图（视觉误导）
- 不标注数值
```

---

### 4. 散点图 (Scatter Plot)

**适用场景：**
- 展示两个变量的关系
- 发现相关性
- 识别异常点

**设计要点：**
```
┌─────────────────────────┐
│               •  •      │
│           •  •  •       │
│         •  •  •         │
│       •  •              │
│     •  •                │
│   •                     │
│ ─────────────────────── │
│   变量X                  │
└─────────────────────────┘

- 添加趋势线
- 标注异常点
- 清晰的坐标轴标签
```

---

### 5. 瀑布图 (Waterfall Chart)

**适用场景：**
- 展示增减变化过程
- 解释数值如何达到最终结果
- 财务分析

**设计要点：**
```
┌─────────────────────────┐
│               ▓▓▓▓▓▓▓▓ │ ← 结束值
│          ░░░░▓▓▓▓▓▓▓▓ │
│     ▓▓▓▓░░░░           │
│ ▓▓▓▓                   │ ← 开始值
│ ─────────────────────── │
│ 初始 +A -B +C 最终      │

▓▓ 正向变化（绿色）
░░ 负向变化（红色）
```

## 图表配色原则

### 配色数量
- 单系列：使用同色系深浅变化
- 多系列：最多 5 种颜色
- 对比：用互补色突出差异

### 色彩语义
```
绿色 → 正向、增长、成功
红色 → 负向、下降、风险
蓝色 → 中性、稳定、专业
黄色 → 警示、注意
灰色 → 背景、辅助信息
```

### 色盲友好
- 避免红绿搭配
- 使用纹理辅助区分
- 考虑蓝橙配色方案

## 数据标签规范

### 显示原则
```
✅ 显示标签的情况：
- 核心数据点
- 关键转折点
- 对比重点

❌ 隐藏标签的情况：
- 数据点过多
- 趋势比数值重要
- 视觉过于拥挤
```

### 格式规范
- 数值：保留合适小数位（通常0-2位）
- 百分比：保留1位小数，如 35.5%
- 大数：使用千分位分隔，如 1,234,567
- 单位：在标题或图例中说明

## 图表设计检查清单

```
□ 图表类型是否合适？
□ 坐标轴标签是否清晰？
□ 图例是否必要且清晰？
□ 数据标签是否恰当？
□ 颜色是否有区分度且语义正确？
□ 是否有不必要的装饰？
□ 标题是否说明图表含义？
□ 数据来源是否标注？
```
