# 波特五力分析

## 概述

波特五力模型（Porter's Five Forces）是哈佛商学院教授迈克尔·波特于1979年提出的行业分析框架，用于评估行业竞争强度和盈利能力。

## 五大竞争力量

```
                    ┌─────────────────────┐
                    │  潜在进入者威胁      │
                    │  (New Entrants)     │
                    └──────────┬──────────┘
                               │
                               ▼
┌──────────────────┐    ┌─────────────────────┐    ┌──────────────────┐
│   供应商议价能力   │───▶│    行业竞争格局      │◀───│   买方议价能力    │
│   (Suppliers)     │    │ (Industry Rivalry)  │    │    (Buyers)      │
└──────────────────┘    └─────────────────────┘    └──────────────────┘
                               │
                               ▼
                    ┌─────────────────────┐
                    │   替代品威胁         │
                    │   (Substitutes)     │
                    └─────────────────────┘
```

## 详细分析

### 1️⃣ 现有竞争者竞争程度

**影响因素：**

| 因素 | 高竞争 | 低竞争 |
|------|--------|--------|
| 竞争者数量 | 多 | 少 |
| 行业增长率 | 低 | 高 |
| 固定成本占比 | 高 | 低 |
| 产品差异化 | 低 | 高 |
| 退出壁垒 | 高 | 低 |

**评估问题：**
- 主要竞争对手是谁？市场地位如何？
- 行业增长速度如何？
- 产品同质化程度如何？
- 价格战频繁吗？

---

### 2️⃣ 潜在进入者威胁

**进入壁垒分析：**

| 壁垒类型 | 说明 | 示例 |
|---------|------|------|
| 规模经济 | 大规模生产降低成本 | 制造业 |
| 品牌认知 | 建立品牌需要时间 | 消费品 |
| 资本需求 | 需要大量初始投资 | 半导体 |
| 转换成本 | 用户更换供应商的成本 | 企业软件 |
| 渠道准入 | 获取销售渠道难度 | 零售业 |
| 政策法规 | 许可证、资质要求 | 金融、医疗 |

**评估问题：**
- 进入这个行业需要多少资金？
- 有什么技术或资质门槛？
- 新进入者能否快速获得客户？

---

### 3️⃣ 替代品威胁

**替代来源：**
- 不同技术路线的产品
- 不同形态的解决方案
- 完全不同的满足方式

**威胁程度取决于：**
- 替代品性价比
- 用户转换成本
- 用户转换意愿

**评估问题：**
- 有什么替代方案可以满足同样的需求？
- 替代品的性价比如何？
- 用户转换到替代品容易吗？

---

### 4️⃣ 供应商议价能力

**强势供应商的特征：**
- 供应商数量少
- 替代供应商少
- 供应商产品差异化高
- 供应商向前整合威胁大
- 采购方数量多且分散

**评估问题：**
- 关键原材料/服务有多少供应商可选？
- 供应商是否可以轻易涨价？
- 是否有替代供应商？

---

### 5️⃣ 买方议价能力

**强势买方的特征：**
- 采购量大
- 产品同质化程度高
- 转换成本低
- 买方向后整合威胁大
- 价格敏感度高

**评估问题：**
- 客户集中度如何？
- 客户是否可以轻易更换供应商？
- 价格谈判谁更有话语权？

---

## 行业吸引力评估

### 五力评分矩阵

```
┌─────────────────────────────────────────────────────────────┐
│                    五力评估矩阵                              │
├─────────────────┬─────────────┬─────────────┬───────────────┤
│     竞争力量     │   弱 (1)    │   中 (2)    │    强 (3)     │
├─────────────────┼─────────────┼─────────────┼───────────────┤
│ 现有竞争程度     │ 盈利空间大  │ 盈利中等    │ 盈利空间小    │
│ 潜在进入威胁     │ 护城河深    │ 护城河中等  │ 护城河浅      │
│ 替代品威胁       │ 替代风险低  │ 替代风险中  │ 替代风险高    │
│ 供应商议价能力   │ 成本可控    │ 成本中等    │ 成本压力大    │
│ 买方议价能力     │ 定价权高    │ 定价权中等  │ 定价权低      │
└─────────────────┴─────────────┴─────────────┴───────────────┘

总分 = 5-15
• 5-7分：高吸引力行业
• 8-11分：中等吸引力
• 12-15分：低吸引力行业
```

---

## 分析步骤

### Step 1: 识别竞争力量
- 列出五种力量的具体表现
- 识别关键参与者

### Step 2: 评估力量强度
- 对每种力量评分（弱/中/强）
- 说明判断依据

### Step 3: 综合判断
- 计算总分
- 识别最关键的力量

### Step 4: 战略建议
- 如何应对每种力量
- 如何建立护城河

## 应用示例

### 案例：云计算行业波特五力分析

**1. 现有竞争程度：中**
- 主要玩家：AWS、Azure、阿里云、腾讯云
- 差异化程度：中（基础服务同质化，增值服务有差异）
- 价格战激烈程度：高

**2. 潜在进入威胁：低**
- 进入壁垒：极高
  - 需要巨额基础设施投资
  - 需要全球数据中心布局
  - 技术壁垒高
- 新进入者多为垂直领域玩家

**3. 替代品威胁：低**
- 替代方案：自建机房、传统IT
- 替代成本高、灵活性差
- 云计算优势明显

**4. 供应商议价能力：中**
- 硬件供应商：议价能力中
- 软件供应商：议价能力低
- 能源供应商：议价能力中

**5. 买方议价能力：高**
- 大客户议价能力强
- 转换成本逐渐降低
- 多云策略普遍

**结论：** 行业竞争激烈，护城河深，盈利能力取决于规模效应

---

## 注意事项

1. **动态视角**：五力随时间变化，需定期更新
2. **行业细分**：细分市场五力可能不同
3. **战略应对**：分析目的是找到应对策略
4. **结合其他框架**：与 SWOT、PEST 配合使用
