# 数据可视化最佳实践

## 图表选择决策树

### 想展示什么？

**比较**
- 项目少 → 柱状图
- 项目多 → 条形图
- 有时间维度 → 折线图 + 数据点

**趋势**
- 单一趋势 → 折线图
- 多个趋势 → 多折线图
- 累计变化 → 面积图

**占比**
- 少于5项 → 饼图
- 5-10项 → 环形图
- 超过10项 → 树状图

**关系**
- 两个变量 → 散点图
- 三个变量 → 气泡图
- 多维度 → 雷达图

**分布**
- 单变量 → 直方图
- 双变量 → 散点图
- 地理分布 → 地图

## 设计规范

### 颜色使用
- 分类数据：使用对比色
- 顺序数据：使用渐变色
- 发散数据：使用双色渐变
- 一致性：同一报告统一配色

### 标签处理
- 数据标签：避免重叠
- 坐标轴：清晰的刻度和单位
- 图例：位置合理，不遮挡数据

### 交互设计（数字报告）
- 鼠标悬停显示详情
- 支持筛选和钻取
- 提供数据下载

## 常见错误

### 数据失真
- ❌ Y轴不从0开始（柱状图）
- ❌ 3D 效果造成误判
- ❌ 面积图比例错误

### 信息过载
- ❌ 一张图太多数据系列
- ❌ 图例过于复杂
- ❌ 背景干扰数据

### 设计问题
- ❌ 颜色对比不足
- ❌ 字体太小
- ❌ 图表比例失调
