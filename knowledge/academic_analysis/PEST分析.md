# PEST 分析法

## 概述

PEST 分析是一种宏观环境分析工具，用于评估 Political（政治）、Economic（经济）、Social（社会）、Technological（技术）四个维度的外部环境因素。

## 四个维度

### 🏛️ Political（政治因素）

政府政策、法规对行业的影响：

**分析要点：**
- 政府政策导向
- 法律法规变化
- 贸易政策和关税
- 政治稳定性
- 税收政策
- 劳动法规

**示例问题：**
- 政府对行业的态度是支持还是限制？
- 有哪些新法规即将实施？
- 国际政治关系对业务有何影响？

### 💰 Economic（经济因素）

宏观经济环境对行业的影响：

**分析要点：**
- GDP 增长率
- 通货膨胀率
- 利率水平
- 汇率变化
- 消费者信心指数
- 行业增长率

**示例问题：**
- 经济周期处于什么阶段？
- 消费者购买力如何变化？
- 融资环境是否宽松？

### 👥 Social（社会因素）

社会文化和人口特征对行业的影响：

**分析要点：**
- 人口结构变化
- 消费观念转变
- 生活方式改变
- 教育水平
- 文化传统
- 健康意识

**示例问题：**
- 目标人群的消费习惯如何变化？
- 社会价值观有哪些新趋势？
- 人口老龄化带来什么影响？

### 🔬 Technological（技术因素）

技术发展对行业的影响：

**分析要点：**
- 新技术突破
- 技术迭代速度
- 研发投入趋势
- 技术转化效率
- 专利申请情况
- 自动化程度

**示例问题：**
- 有哪些颠覆性技术正在出现？
- 技术迭代速度是否加快？
- 技术壁垒是提高还是降低？

## 分析步骤

### Step 1: 确定分析范围
- 明确分析对象（行业/企业/项目）
- 确定分析时间跨度

### Step 2: 信息收集
- 收集政策文件
- 研究经济数据
- 分析社会报告
- 跟踪技术动态

### Step 3: 因素分析
- 逐维度列出关键因素
- 评估每个因素的影响程度
- 判断是机会还是威胁

### Step 4: 形成洞察
- 总结关键环境趋势
- 识别机会和威胁
- 为战略决策提供依据

## PEST 与 SWOT 的结合

```
PEST 分析结果 → 填充 SWOT 的 O 和 T

┌─────────────────────────────────────────┐
│            PEST 分析                    │
├─────────────────────────────────────────┤
│ P → 政策利好 = 机会 (O)                 │
│ E → 经济下行 = 威胁 (T)                 │
│ S → 消费升级 = 机会 (O)                 │
│ T → 技术替代 = 威胁 (T)                 │
└─────────────────────────────────────────┘
                    ↓
┌─────────────────────────────────────────┐
│            SWOT 分析                    │
├─────────────────────────────────────────┤
│         │ 机会 (O)    │ 威胁 (T)        │
│ 优势 (S)│ SO策略      │ ST策略          │
│ 劣势 (W)│ WO策略      │ WT策略          │
└─────────────────────────────────────────┘
```

## 扩展版本：PESTEL

PESTEL 在 PEST 基础上增加两个维度：

- **Environmental（环境）**：气候变化、环保法规、可持续发展
- **Legal（法律）**：具体法律法规、知识产权保护、消费者权益

## 应用示例

### 案例：电动汽车行业 PEST 分析

**政治因素 (P)**
- 🟢 新能源汽车补贴政策
- 🟢 碳中和目标推动
- 🟡 贸易摩擦影响电池进口

**经济因素 (E)**
- 🟢 油价上涨提升电动车吸引力
- 🟡 经济增速放缓影响消费
- 🟢 电池成本持续下降

**社会因素 (S)**
- 🟢 环保意识增强
- 🟢 年轻人更接受新技术
- 🟡 充电焦虑仍存在

**技术因素 (T)**
- 🟢 电池技术突破
- 🟢 自动驾驶技术进步
- 🟡 充电基础设施仍需完善

**结论：** 行业整体处于政策+技术双轮驱动的上升期

---

## 注意事项

1. **聚焦关键因素**：每个维度聚焦 3-5 个最关键因素
2. **保持更新**：环境变化快，定期更新分析
3. **关联分析**：维度之间存在相互影响
4. **结合行业**：根据行业特点调整分析重点
