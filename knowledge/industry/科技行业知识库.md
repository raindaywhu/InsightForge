# 科技行业知识库

> 科技行业分析框架、细分赛道、技术趋势与投资逻辑

---

## 1. 行业概况

### 1.1 行业定义
科技行业是指以技术创新为核心驱动力的行业，包括半导体、软件、互联网、人工智能、云计算等细分领域。

### 1.2 产业链结构
```
基础层：芯片设计、制造、封装测试
平台层：操作系统、云平台、中间件
应用层：软件应用、互联网服务、智能终端
```

### 1.3 市场规模
- 全球科技市场：~5 万亿美元
- 中国科技市场：~20 万亿人民币
- 主要细分市场占比：
  - 半导体：25%
  - 软件：30%
  - 互联网：30%
  - 其他：15%

## 2. 关键分析框架

### 2.1 半导体行业分析框架

**产业链环节**：
| 环节 | 说明 | 主要玩家 |
|------|------|---------|
| EDA | 芯片设计工具 | Synopsys、Cadence |
| 设计 | 芯片架构设计 | NVIDIA、高通、海思 |
| 制造 | 晶圆代工 | TSMC、三星、中芯国际 |
| 封测 | 封装测试 | 日月光、长电科技 |
| 设备 | 制造设备 | ASML、应用材料 |
| 材料 | 晶圆、化学品 | 信越化学、JSR |

**核心指标**：
- 毛利率：设计 50%+，制造 30-50%，封测 15-25%
- 库存周转天数：需求周期判断
- 产能利用率：景气度指标
- 资本开支：扩产节奏

**技术节点**：
- 先进制程：< 7nm（EUV 光刻）
- 成熟制程：28-90nm（汽车、MCU）
- 特色工艺：功率、模拟、存储

### 2.2 软件行业分析框架

**商业模式**：
- SaaS：订阅收费、年费/月费
- License：一次性授权
- 服务费：定制开发、维护
- 免费+增值：基础免费、高级付费

**关键指标**：
| 指标 | 说明 | 标准 |
|------|------|------|
| ARR | 年度经常性收入 | 核心增长指标 |
| NRR | 净收入留存率 | > 100% 为优 |
| LTV/CAC | 客户价值/获客成本 | > 3 为健康 |
| Churn Rate | 客户流失率 | < 10% 为优 |
| Gross Margin | 毛利率 | > 70% |

**产品类型**：
- 通用软件：ERP、CRM、办公软件
- 垂直软件：金融、医疗、教育
- 开发工具：IDE、低代码、DevOps
- 基础软件：数据库、中间件、安全

### 2.3 互联网行业分析框架

**业务模式**：
- 电商：交易佣金、广告、增值服务
- 社交：广告、虚拟商品、会员
- 内容：订阅、广告、打赏
- 游戏：付费下载、内购、订阅

**流量指标**：
- DAU/MAU：日活/月活用户
- 用户时长：人均使用时间
- 留存率：次日/7日/30日
- ARPU：人均收入
- 转化率：浏览→购买/注册

**盈利模式**：
| 模式 | 代表企业 | 关键指标 |
|------|---------|---------|
| 广告 | 字节、百度 | 广告加载率、eCPM |
| 电商 | 阿里、拼多多 | GMV、货币化率 |
| 会员 | 腾讯视频、爱奇艺 | 会员数、续费率 |
| 游戏 | 腾讯、网易 | ARPU、付费率 |

### 2.4 人工智能分析框架

**技术分类**：
- 计算机视觉：图像识别、视频分析
- 自然语言处理：对话、翻译、写作
- 机器学习：推荐、预测、决策
- 知识图谱：知识抽取、推理

**应用场景**：
| 场景 | 应用 | 代表企业 |
|------|------|---------|
| 自动驾驶 | 感知、决策 | 特斯拉、Waymo |
| 智能客服 | 对话机器人 | 各大云厂商 |
| 内容生成 | AIGC | OpenAI、Midjourney |
| 推荐系统 | 个性化推荐 | 字节、阿里 |

**核心指标**：
- 模型性能：准确率、F1、BLEU
- 推理延迟：响应时间
- 训练成本：算力、数据量
- 商业化：收入、客户数

## 3. 技术趋势

### 3.1 前沿技术
- **生成式 AI**：GPT、Stable Diffusion、Sora
- **大模型**：GPT-4、Claude、Gemini、GLM
- **边缘计算**：端侧 AI、物联网
- **量子计算**：量子优势、密码学影响
- **Web3**：区块链、NFT、DeFi

### 3.2 产业趋势
- **国产替代**：芯片、操作系统、数据库
- **云原生化**：容器、微服务、DevOps
- **安全合规**：数据安全、隐私计算
- **降本增效**：自动化、智能化

### 3.3 投资热点
- AI 基础设施：算力、存储、网络
- AI 应用：Agent、AIGC、垂直 SaaS
- 硬件创新：AR/VR、可穿戴、机器人
- 企业服务：安全、协作、低代码

## 4. 分析模板

### 4.1 半导体企业分析模板

```markdown
## XX 半导体企业分析报告

### 一、公司概况
- 基本信息、核心团队、股权结构

### 二、业务分析
- 产品线：芯片类型、应用领域
- 技术能力：制程节点、工艺水平
- 客户结构：主要客户、应用占比

### 三、市场分析
- 市场规模与增速
- 竞争格局
- 国产替代空间

### 四、财务分析
- 收入结构
- 盈利能力（毛利率、净利率）
- 库存与产能

### 五、估值与建议
- 估值方法（PS、PE、EV/EBITDA）
- 风险提示（技术、客户、周期）
- 投资建议
```
