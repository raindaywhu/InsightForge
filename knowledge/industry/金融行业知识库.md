# 金融行业知识库

> 金融行业分析框架、关键指标、监管政策与市场趋势

---

## 1. 行业概况

### 1.1 行业定义
金融行业是指经营金融商品的特殊行业，包括银行业、证券业、保险业、信托业、基金业等。

### 1.2 产业链结构
```
上游：金融基础设施（交易所、清算机构、支付系统）
中游：金融机构（银行、券商、保险、基金）
下游：金融服务（个人金融、企业金融、政府金融）
```

### 1.3 市场规模
- 全球金融市场规模：~500 万亿美元
- 中国金融市场规模：~350 万亿人民币
- 主要细分市场占比：
  - 银行业：60%
  - 证券业：20%
  - 保险业：15%
  - 其他：5%

## 2. 关键分析框架

### 2.1 银行业分析框架

**财务指标**：
| 指标 | 说明 | 健康标准 |
|------|------|---------|
| 不良贷款率 | 不良贷款/总贷款 | < 2% |
| 拨备覆盖率 | 拨备/不良贷款 | > 150% |
| 资本充足率 | 资本/风险加权资产 | > 12% |
| 净息差 | 利息收入-利息支出/生息资产 | > 2% |
| ROE | 净利润/净资产 | > 10% |

**风险指标**：
- 信用风险：不良率、逾期率
- 市场风险：VaR、压力测试
- 流动性风险：LCR、NSFR
- 操作风险：损失事件频率

### 2.2 证券业分析框架

**业务结构**：
- 经纪业务：佣金收入、市占率
- 投行业务：承销规模、IPO 数量
- 资管业务：AUM、主动管理占比
- 自营业务：投资收益率、风险敞口

**监管指标**：
- 风险覆盖率 > 100%
- 资本杠杆率 > 8%
- 流动性覆盖率 > 100%

### 2.3 保险业分析框架

**核心指标**：
| 指标 | 说明 | 标准 |
|------|------|------|
| 综合成本率 | 赔付+费用/保费 | < 100% |
| 新业务价值 | 新业务未来利润现值 | 年增长 |
| 内含价值 | 净资产+有效业务价值 | 年增长 |
| 偿付能力充足率 | 实际资本/最低资本 | > 150% |

## 3. 监管政策

### 3.1 主要监管机构
- **中国**：央行（PBOC）、银保监会、证监会
- **美国**：美联储（Fed）、SEC、FDIC
- **欧洲**：ECB、ESMA
- **国际**：BIS、FSB、IMF

### 3.2 近期监管重点
- **宏观审慎**：系统性风险监测、逆周期调节
- **金融科技**：数据安全、算法监管、沙盒机制
- **ESG**：绿色金融标准、气候风险披露
- **反洗钱**：KYC、可疑交易报告

### 3.3 合规要求
- 资本充足率要求
- 风险管理制度
- 内部控制体系
- 信息披露义务
- 消费者保护

## 4. 市场趋势

### 4.1 数字化转型
- **开放银行**：API 生态、场景金融
- **智能投顾**：算法驱动的财富管理
- **数字货币**：CBDC、稳定币、DeFi
- **区块链**：跨境支付、供应链金融

### 4.2 业务创新
- 普惠金融：小微企业、农村金融
- 绿色金融：碳中和债券、ESG 投资
- 场景金融：消费金融、供应链金融
- 财富管理：家族办公室、私人银行

### 4.3 风险挑战
- 利率风险：利率市场化、息差收窄
- 信用风险：经济下行、债务违约
- 科技风险：网络安全、系统故障
- 合规风险：监管趋严、处罚加重
